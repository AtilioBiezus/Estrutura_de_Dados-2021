Informações

  • PILHA ou HEAP
  • É uma estrutura de dados simples.
  • Principal ponto de uma pilha é que todo o acesso a seus elementos são feitos através do seu topo.
  • Pilha ou FILO (First-In, Last-Out) significa o primeiro que entra é o último que sai.
  • Uma pilha é uma estrutura de dados que contém uma lista de elementos
  • Uma pilha possui duas operações principais que ocorrem apenas no topo da pilha, que são push e pop.
